SOP – Team Performance Monitoring

  • Red (0–60%) → Critical Risk → Must go into PIP/Firing track.
  • Green (61–90%) → Performing → Stable; sustain performance.
  • Purple (91%+) → High Performer → Recognition + growth opportunities.

Every Day:

  • ASM → SRs: Daily check-in with all SRs (field review, calls, or messages).
  • RSM → ASMs: Daily check-in to ensure ASM reviews are happening.
  • Week 1 – Flag & Escalate → SR/ASM in Red/Yellow flagged immediately; HR notified.
  • Week 2 – PIP Initiation → Individual placed on PIP with clear targets, assigned trainer, documented training sessions.
  • Week 3 – Review → Progress reviewed by ASM + RSM + HR; if partial improvement → focused training.
  • Week 4 – 2nd Review → Follow-up review by ASM + RSM + HR; if still partial → additional training + Joint Working with ASM.
  • Week 5 – Final Review & Exit → If still unsatisfactory → Exit; if improved → Return to normal monitoring.
  1. Attendance Time: 9:30 AM for All Zones
  2. First Call: 10:30 AM for Guwahati & Shillong; 11:00 AM for rest
  3. TC: 40/day for Guwahati & Shillong; 30/day for rest
  4. PC: 32/day for Guwahati & Shillong; 21/day for rest
  5. Time on Field: 6 Hrs/day
  6. Distributor Search or Visit: Min 3/day and KYC must be filled (FA App: Official Work → Distributor Search)

Rule:

  • If Actual ≥ Target (TGT) → Score = 100% and mark = “+”.
  • If Actual < Target (TGT) → Score = (Actual ÷ Target) × 100 and mark = “–”.

Example: Target = 100, Actual = 80 → Score = 80% (marked “–”).
